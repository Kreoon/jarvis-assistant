### SECCIÓN 3: ANÁLISIS ESTRATÉGICO — 12 DIMENSIONES
*Aplicando: Cialdini, Maslow, DISC, Schwartz, Cerebro Triuno, RICE*

**3.1 ESTRUCTURA**

```
HOOK (0-3 segundos)                              Score: 8/10
────────────────────────────────────────────────────────────
Técnica: Metáfora disruptiva + Pattern interrupt
"La vaca no da leche" — rompe una creencia asumida

Efecto Zeigarnik: SI — abre loop (¿entonces qué pasa?)
Hooks comparables: "Nadie te dice esto sobre [tema]"

> Referencia Gemini: Primer plano directo a cámara, texto
> grande superpuesto, expresión seria que genera expectativa
```

```
DESARROLLO                                       Retención: 7/10
────────────────────────────────────────────────────────────
Estructura PASTOR detectada:
  P - Problema: "la gente espera que las cosas lleguen"
  A - Amplificar: "nadie te lo dice pero..."
  S - Story: metáfora de la vaca
  T - Transformación: "cambia tú"
  O - Oferta: libro gratis
  R - Respuesta: "comenta VERDAD"

Técnicas de retención: revelación progresiva, tono crescendo
Punto débil: 65s largo para Reels — ideal 30-45s
```

```
CTA / CIERRE                                     Efectividad: 8/10
────────────────────────────────────────────────────────────
Tipo: Doble CTA
  1. Engagement: "Comenta VERDAD" (micro-compromiso)
  2. Lead magnet: libro gratis en bio

Cialdini - Compromiso/Coherencia: SI
  Micro-sí detectado: pide comentario antes del lead magnet

> Referencia Gemini: primer plano final, tono imperativo,
> señala hacia abajo (bio)
```

```
FORMATO                                          Optimización: 7/10
────────────────────────────────────────────────────────────
Formato actual: Reel 65s, talking head
¿Óptimo?: NO — demasiado largo para el mensaje
Ideal: 30-40s más compacto

Repurposing posible:
  → 3 clips de 15s para Stories
  → 1 carrusel "5 metáforas de acción"
  → 1 post de texto con la metáfora
  → 1 thread en X con la frase + desarrollo
  → 1 YouTube Short versión compacta
```

**3.2 COPY**

```
FÓRMULA DETECTADA: AIDA + Revelación Metafórica
────────────────────────────────────────────────────────────
A - Atención: "La vaca no da leche" (metáfora)
I - Interés: "hay que ordeñarla" (verdad oculta)
D - Deseo: "todo requiere acción" (transformación)
A - Acción: "comenta VERDAD" + lead magnet
```

```
PALABRAS DE PODER
────────────────────────────────────────────────────────────
  nadie te lo ha dicho    →  Exclusividad + Curiosidad
  verdad                  →  Autoridad + Contraste
  cambiar                 →  Transformación
  ordeñar                 →  Visual + Sensorial
  todo lo que quieres     →  Deseo universal

Neurocopy aplicado:
  [+] Negativo en hook ("no da") — 2x más impactante
  [+] Palabra sensorial ("ordeñar")
  [-] No usa números impares
  [-] Sin contraste temporal ("en X días")
```

```
GATILLOS MENTALES (Cialdini + Neuroventas)
────────────────────────────────────────────────────────────
  [SI] Reciprocidad      — Ofrece libro gratis
  [SI] Compromiso        — Pide comentar antes de pedir
  [SI] Autoridad         — Tono de "yo sé algo que tú no"
  [SI] Escasez implícita — "nadie te lo dice"
  [NO] Urgencia          — Sin deadline ni límite de tiempo
  [NO] Prueba social     — Sin testimonios ni números
  [NO] Contraste/Anclaje — Sin antes/después
  [NO] FOMO              — Sin "te vas a perder..."

  Dolor vs Placer: Lidera con DOLOR (pasividad)
  Regla neuroventas: CORRECTO — dolor motiva 2.5x más
```

```
TONO Y VOZ
────────────────────────────────────────────────────────────
Tono: Directo, maternal, empoderador
Cerebro triuno:
  Reptiliano (20%): amenaza sutil ("estás esperando")
  Límbico (50%): emoción, historia, conexión ← DOMINANTE
  Neocórtex (30%): lógica de la metáfora

  Regla: Vende al límbico, justifica al neocórtex — CUMPLE

DISC target: Dirigido a perfil I (Influyente)
  → Usa storytelling y emoción
  → Podría agregar datos para perfil C (Concienzudo)
```

**3.3 ESTRATEGIA**

```
POSICIÓN EN EMBUDO
────────────────────────────────────────────────────────────
TOFU ████████████████████  ← ESTE CONTENIDO
MOFU ░░░░░░░░░░░░░░░░░░░░
BOFU ░░░░░░░░░░░░░░░░░░░░

Objetivo: Alcance + primeros leads (comentarios → libro)

Schwartz - Nivel de conciencia: 2-3
  Saben que tienen un problema (pasividad)
  No conocen la solución específica
  → Correcto: el contenido EDUCA sobre la solución
```

```
PILAR DE CONTENIDO
────────────────────────────────────────────────────────────
Distribución ideal:  Educar 40% | Entretener 25% | Inspirar 20% | Vender 15%
Este contenido:      Educar 70% | Inspirar 30%

  → Desbalanceado pero funciona para TOFU
  → Oportunidad: agregar elemento de entretenimiento
    (humor, meme, reference) para amplificar alcance

Pecado capital activado: PEREZA
  → "Deja de esperar" apela a la pereza/comodidad
  → Muy efectivo como motivador de acción
```

```
ÁNGULO DE VENTA
────────────────────────────────────────────────────────────
Pain point:     Pasividad, victimismo, esperar resultados sin actuar
Deseo:          Control, autonomía, hacer que las cosas pasen
Transformación: "De pasivo que espera → a activo que ordeña"

Maslow: AUTORREALIZACIÓN
  → Apunta al nivel más alto (propósito, acción, legado)
  → Regla: vende al nivel donde ESTÁ la audiencia
  → Riesgo: si la audiencia está en Seguridad (pagar cuentas),
    este mensaje puede sentirse desconectado
```

```
VIRALIDAD                                        Score: 9/10
────────────────────────────────────────────────────────────
Fórmula: (Emoción x Utilidad x Timing) / Fricción
         (9 × 7 × 8) / 2 = alto potencial viral

  Relatabilidad:      ALTA — todos conocen la pasividad
  Shareability:       ALTA — "envía esto al que necesita oírlo"
  Save-ability:       MEDIA — no tiene datos para guardar
  Polémica:           BAJA-MEDIA — controlada, no tóxica
  Novedad:            MEDIA — metáfora conocida pero bien ejecutada

  Emoción dominante:  Revelación / Despertar
  Ranking emocional:  Asombro (7) + Inspiración (8)

  Patrón viral:       "Nadie te dice esto pero..."
  → Top 3 patrones más virales de 2025-2026
```

### SECCIÓN 6: GUÍA DE PRODUCCIÓN

**6.1 Checklist pre-grabación**
```
[ ] Espacio silencioso con buena luz natural o ring light
[ ] Teléfono cargado, modo avión
[ ] Script memorizado (no leído)
[ ] Outfit que comunique el mensaje (profesional pero accesible)
[ ] Fondo limpio o contexto relevante (oficina, calle, etc.)
[ ] App de grabación lista (Instagram/cámara nativa)
[ ] Texto en pantalla preparado (CapCut/InShot)
[ ] Música seleccionada (si aplica)
```
